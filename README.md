# nixos_config

A NixOS configuration using flakes and home-manager.

## Switching to This Config

### Prerequisites

- NixOS installed with flakes enabled (default in recent NixOS)
- Git installed

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/nixos_config.git /etc/nixos
   cd /etc/nixos
   ```

2. **Review and customize the configuration:**
   
   Edit `hosts/grajpap/default.nix` to set your hostname:
   ```nix
   # Change "grajpap" to your hostname
   ```
   
   Edit `hosts/grajpap/hardware-configuration.nix` to match your hardware (generated by `nixos-generate-config`).

   Edit `hosts/grajpap/default.nix` to update:
   - User username
   - Timezone
   - GPU drivers if needed

3. **Update the flake inputs:**
   ```bash
   nix flake update
   ```

4. **Apply the configuration:**
   ```bash
   sudo nixos-rebuild switch --flake .#yourhostname
   ```

### Project Structure

```
.
├── flake.nix                  # Flake entry point
├── configuration.nix          # Shared system config
├── hosts/
│   └── grajpap/               # Host-specific config
│       ├── default.nix         # Host module
│       └── hardware-configuration.nix
├── home/                      # Home-manager config
├── system/                    # System modules
└── theme/                     # Stylix theming
```

### Available Commands

```bash
# Build the system
sudo nixos-rebuild switch --flake .#grajpap

# Build the ISO (for installation)
nix build .#packages.x86_64-linux.grajpap-iso

# Update the system
sudo nixos-rebuild switch --upgrade --flake .#grajpap
```

### Using nh (NixOS Helper)

If you have `nh` installed:
```bash
nh os switch
```

### Keeping Updated

```bash
cd /etc/nixos
git pull
sudo nixos-rebuild switch --flake .#yourhostname
```
